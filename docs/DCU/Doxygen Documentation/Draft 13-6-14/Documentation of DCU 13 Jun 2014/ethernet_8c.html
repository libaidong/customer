Detailed Description

Provides set of APIs to perform operation on socket.

Provides set of APIs to perform operation on socket. APIs are to open, bind and connect to socket. It provide support for both UDP and TCP.

unsigned char  IsSecured,
SSL_CTX **  Ctx 
)

API to open socket. Socket can be UDP or TCP. Provision is to be specify whether socket is for server or whether it is blocking.

Parameters
IPLocal IP address to bind on.
PortLocal port address to bind on.
ProtocolTransport layer protocol. It can be either TCP or UDP.
IsServerSpecifies whether the socket will act as server or client. Applicable on for TCP.
IsBlockingSpecifies whether the socket is blocking or non blocking.
IsSecuredSpecifies whether the socket connection is SSL secured or not.
CtxSpecifies context of SSL.
Returns
-1 in case failure in socket opening. -2 in case failure in setting socket option for reusing the address for TCP. -3 in case failure in binding to socket. -4 in case failure in getting socket option for non blocking. -5 in case failure in setting socket option for non blocking. -6 in case API fail to set the listen queue for TCP. Socket fd in case of success.
