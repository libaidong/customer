int StorePoolRefOfDeviceForBMS ( POLLING_INFO *  PollingRef,
DEVICE_INFO *  DeviceRef 
)

brief API used to store reference of pool which holds data / status of the device. API search for BMS register form BMS Info, if it is present in the BMSinfo then only store the reference otherwise return error value.

Parameters
PollingRefSpecifies reference to polling info list for the device for which BMS is to be allocated.
DeviceRefSpecifies reference to the device info list.
Returns
Returns 1 as success if valid BMS register entry present in BMS info otherwise return -1 value as error.
