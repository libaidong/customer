Detailed Description

Header fle for ethernet application file. Provides set of APIs to perform application level handling for ethernet communication.

Provides set of APIs to perform application level handling for ethernet communication. Handles connection establishment and reading logic. Most of the APIs will be running as thread.

void* TCPClient ( void *  Info)

API to handle the TCP client. It include connection, reading bytes and informing upper layer. If shared connection and interface are correct then this function will stay in loop.

Parameters
InfoReference index of connections.
Returns
NULL in case invalid connection reference. NULL in case the interface mentioned in connection is invalid. Else this function stay in loop.
