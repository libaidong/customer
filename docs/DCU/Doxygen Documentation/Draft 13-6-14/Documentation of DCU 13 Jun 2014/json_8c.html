Detailed Description

For handling the JSON communication for SCS and PC utility.

For handling the JSON communication for SCS and PC utility. It peforms activity on JSON one at a time e.g. file upload / download, pushing device details, setting value of IO etc. Same state machine is used for both SCS and PC utility, it state machine behaves based on activity which is set.
