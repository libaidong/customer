API to add the entry of connection. The must be single connection for each logical and physical interface.

Parameters
InfTagUnique tag to identify the interface.
CommInfoInformation required for communication for the corresponding connection.
BufferSizeSize of buffer used for bytes transfer which are received on connection.
CBPtrPointer to callback function to process the events and received bytes.
ThreadPtrPointer to function which will handle the connection and read bytes on connection.
NameName of the connection, used to identify the connection. Useful in logs for multithread.
DeviceRefReference of the device entry for which the connection is to be created.
Returns
On success it returns the index of the Connection array. In case of invalid tag -1 is returned. In case of failure in creation of thread -2 is returned.
