Detailed Description

Header file for interface handler. Provide APIs related to communication interfaces.

Provide APIs related to communication interfaces. As of now supports ETH, Serial and GPRS interface. APIs for registering interface, monitor ETH interface and initializing and monitoring GPRS interface.
